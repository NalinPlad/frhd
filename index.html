F R H D - P Y T H O N

A library to edit FRHD tracks.


































































WHAT IT IS




FRHD-PYTHON IS A PYTHON 3.x LIBRARY TO EDIT TRACKS FROM THE POPULAR HTML5 GAME FREERIDERHD (FRHD).
IT CONVERTS USER INPUT TO BASE32, WHICH IS THE CODE THAT FRHD USES TO DISPLAY THE USER-GENERATED TRACKS.
IT CAN CURRENTLY CREATE TWO TYPES OF LINES, ALL THE CURRENT POWER-UPS MINUS THE TELEPORT, AS WELL AS MOVE
TRACKS AROUND HIGHLY PRECISELY. THERE ARE SEVERAL NEW FEATURES, SUCH AS AUTOMATIC TRACK GENERATION AND
THE ADDITION OF VEHICLES AND THE TELEPORT.
















AS IT IS STILL IN ACTIVE DEVELOPMENT, UPDATES ARE BEING PUSHED CONSTANTLY. THE SYNTAX IS STRAIGHTFORWARD AND EASY TO USE FOR BEGINNERS.
